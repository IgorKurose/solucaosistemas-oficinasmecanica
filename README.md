# Frontend Landing Page (Vercel)

Projeto reduzido para frontend puro (Vite + React), pronto para deploy na Vercel.

## Arquivo principal da tela

- `src/pages/landing.tsx`

## Estrutura

- `src/main.tsx`: ponto de entrada
- `src/App.tsx`: renderiza a landing
- `src/index.css`: estilos globais
- `public/assets/*`: imagens usadas pela landing

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
